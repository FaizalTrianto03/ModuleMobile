# Modul 2: Network API & Web Interfaces

## 1. Pengantar HTTP & REST API

Aplikasi mobile modern sering membutuhkan komunikasi dengan server melalui API. **REST API** adalah standar komunikasi berbasis HTTP yang umum digunakan.

- **HTTP Methods**: GET, POST, PUT, DELETE
- **JSON**: Format data yang sering digunakan untuk pertukaran data

---

## 2. Melakukan Request HTTP di Flutter

### 2.1. Menggunakan Package `http`

Tambahkan pada `pubspec.yaml`:

```yaml
dependencies:
  http: ^0.13.5
```

#### Contoh Request GET

```dart
import 'package:http/http.dart' as http;
import 'dart:convert';

void fetchData() async {
  final response = await http.get(Uri.parse('https://jsonplaceholder.typicode.com/posts/1'));
  if (response.statusCode == 200) {
    var data = json.decode(response.body);
    print(data);
  } else {
    print('Gagal mengambil data');
  }
}
```

---

### 2.2. Menggunakan GetConnect (GetX)

GetX menyediakan `GetConnect` untuk networking yang lebih mudah.

Tambahkan pada `pubspec.yaml`:

```yaml
dependencies:
  get: ^4.6.5
```

#### Contoh GetConnect

```dart
import 'package:get/get.dart';

class ApiProvider extends GetConnect {
  Future<Response> getPost(int id) => get('https://jsonplaceholder.typicode.com/posts/$id');
}
```

## 3. Parsing JSON di Flutter

### 3.1. Model Data

```dart
class Post {
  final int id;
  final String title;
  final String body;

  Post({required this.id, required this.title, required this.body});

  factory Post.fromJson(Map<String, dynamic> json) {
    return Post(
      id: json['id'],
      title: json['title'],
      body: json['body'],
    );
  }
}
```

### 3.2. Menggunakan Model

```dart
void fetchAndParse() async {
  final response = await http.get(Uri.parse('https://jsonplaceholder.typicode.com/posts/1'));
  if (response.statusCode == 200) {
    var data = json.decode(response.body);
    Post post = Post.fromJson(data);
    print(post.title);
  }
}
```

## 4. Studi Kasus: Menampilkan Data dari API

### 4.1. Controller GetX

```dart
import 'package:get/get.dart';
import 'package:http/http.dart' as http;
import 'dart:convert';
import 'post.dart';

class PostController extends GetxController {
  var post = Rxn<Post>();
  var isLoading = false.obs;

  void fetchPost(int id) async {
    isLoading.value = true;
    final response = await http.get(Uri.parse('https://jsonplaceholder.typicode.com/posts/$id'));
    if (response.statusCode == 200) {
      post.value = Post.fromJson(json.decode(response.body));
    }
    isLoading.value = false;
  }
}
```

### 4.2. UI Menampilkan Data

```dart
import 'package:flutter/material.dart';
import 'package:get/get.dart';
import 'post_controller.dart';

class PostPage extends StatelessWidget {
  final PostController c = Get.put(PostController());

  PostPage({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('Post dari API')),
      body: Center(
        child: Obx(() {
          if (c.isLoading.value) {
            return const CircularProgressIndicator();
          }
          if (c.post.value == null) {
            return ElevatedButton(
              onPressed: () => c.fetchPost(1),
              child: const Text('Ambil Data'),
            );
          }
          return Column(
            mainAxisAlignment: MainAxisAlignment.center,
            children: [
              Text('ID: ${c.post.value!.id}'),
              Text('Judul: ${c.post.value!.title}'),
              Text('Isi: ${c.post.value!.body}'),
            ],
          );
        }),
      ),
    );
  }
}
```

## 5. Error Handling & Best Practice

- Selalu cek status code response.
- Gunakan try-catch untuk menangani error jaringan.
- Pisahkan model, controller, dan UI.
- Gunakan loading indicator saat request berlangsung.
