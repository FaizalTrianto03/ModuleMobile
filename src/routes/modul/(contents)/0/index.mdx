## 1. Pendahuluan

Flutter adalah framework open-source dari Google untuk membangun aplikasi mobile, web, dan desktop dengan satu basis kode. Dengan Flutter, Anda dapat membuat aplikasi yang indah, cepat, dan multiplatform. Pada modul ini, Anda akan belajar cara menyiapkan lingkungan pengembangan Flutter menggunakan FVM (Flutter Version Manager) dan mengenal dasar-dasar Flutter serta GetX sebagai state management.

---

## 2. Persiapan Lingkungan Pengembangan

### 2.1. Instalasi FVM (Flutter Version Manager)

FVM memudahkan pengelolaan berbagai versi Flutter dalam satu mesin.

**Langkah-langkah:**

1. Pastikan sudah terinstal [Git](https://git-scm.com/) dan [Dart SDK](https://dart.dev/get-dart).
2. Instal FVM menggunakan pub:
   ```bash
   dart pub global activate fvm
   ```
3. Tambahkan FVM ke PATH:
   - Cek lokasi pub global:
     ```bash
     dart pub global list
     ```
   - Tambahkan ke PATH (contoh):
     ```bash
     export PATH="$PATH":"$HOME/.pub-cache/bin"
     ```

### 2.2. Instalasi Flutter via FVM

1. Instal Flutter stable:
   ```bash
   fvm install stable
   ```
2. Atur default Flutter:
   ```bash
   fvm global stable
   ```
3. Cek versi Flutter:
   ```bash
   fvm flutter --version
   ```

### 2.3. Instalasi IDE

- **Rekomendasi:** [Visual Studio Code](https://code.visualstudio.com/) atau [Android Studio](https://developer.android.com/studio)
- Instal plugin Flutter & Dart di IDE.

---

## 3. Pengenalan Flutter

- **Flutter** menggunakan bahasa pemrograman Dart.
- **Widget** adalah elemen dasar UI di Flutter.
- **Hot Reload** memudahkan pengembangan aplikasi secara cepat.

---

## 4. Pengenalan GetX

GetX adalah package Flutter untuk state management, dependency injection, dan route management yang sederhana dan efisien.

**Instalasi GetX:**
Tambahkan pada `pubspec.yaml`:

```yaml
dependencies:
  get: ^4.6.5
```

---

## 5. Membuat Proyek Flutter Pertama

1. Buat proyek baru:
   ```bash
   fvm flutter create hello_flutter
   cd hello_flutter
   ```
2. Jalankan aplikasi:
   ```bash
   fvm flutter run
   ```

---

## 6. Struktur Proyek Flutter

```
hello_flutter/
├── android/
├── ios/
├── lib/
│   └── main.dart
├── pubspec.yaml
└── ...
```

- **lib/main.dart**: Titik masuk aplikasi.
- **pubspec.yaml**: Konfigurasi dependencies.

---

## 7. Contoh Kode Flutter Sederhana

```dart
import 'package:flutter/material.dart';

void main() {
  runApp(const MyApp());
}

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Hello Flutter',
      home: Scaffold(
        appBar: AppBar(
          title: const Text('Hello Flutter'),
        ),
        body: const Center(
          child: Text('Selamat datang di Flutter!'),
        ),
      ),
    );
  }
}
```

- Manfaatkan Hot Reload untuk mempercepat pengembangan.
