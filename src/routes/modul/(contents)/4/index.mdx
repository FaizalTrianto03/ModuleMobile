# Modul 4: Sensor-driven Computing

## 1. Pengantar Sensor di Perangkat Mobile

Perangkat mobile modern dilengkapi berbagai sensor, seperti:

- **Accelerometer**: Mengukur percepatan gerak perangkat.
- **Gyroscope**: Mengukur rotasi perangkat.
- **Proximity**: Mendeteksi jarak objek ke perangkat.
- **Light Sensor**: Mengukur intensitas cahaya.

Sensor digunakan untuk berbagai aplikasi seperti fitness tracker, game, navigasi, dsb.

---

## 2. Mengakses Sensor di Flutter

### 2.1. Instalasi Package `sensors_plus`

Tambahkan pada `pubspec.yaml`:

```yaml
dependencies:
  sensors_plus: ^4.0.2
  get: ^4.6.5
```

### 2.2. Membaca Data Accelerometer

```dart
import 'package:flutter/material.dart';
import 'package:sensors_plus/sensors_plus.dart';

class AccelerometerPage extends StatefulWidget {
  const AccelerometerPage({super.key});

  @override
  State<AccelerometerPage> createState() => _AccelerometerPageState();
}

class _AccelerometerPageState extends State<AccelerometerPage> {
  double x = 0, y = 0, z = 0;

  @override
  void initState() {
    super.initState();
    accelerometerEvents.listen((AccelerometerEvent event) {
      setState(() {
        x = event.x;
        y = event.y;
        z = event.z;
      });
    });
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('Accelerometer')),
      body: Center(
        child: Text('x: $x\ny: $y\nz: $z', style: const TextStyle(fontSize: 24)),
      ),
    );
  }
}
```

## 3. Integrasi Sensor dengan GetX

### 3.1. Controller Sensor

```dart
import 'package:get/get.dart';
import 'package:sensors_plus/sensors_plus.dart';

class SensorController extends GetxController {
  var x = 0.0.obs;
  var y = 0.0.obs;
  var z = 0.0.obs;

  @override
  void onInit() {
    super.onInit();
    accelerometerEvents.listen((event) {
      x.value = event.x;
      y.value = event.y;
      z.value = event.z;
    });
  }
}
```

### 3.2. UI dengan GetX

```dart
import 'package:flutter/material.dart';
import 'package:get/get.dart';
import 'sensor_controller.dart';

class SensorPage extends StatelessWidget {
  final SensorController c = Get.put(SensorController());

  SensorPage({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('Sensor dengan GetX')),
      body: Center(
        child: Obx(() => Text(
              'x: ${c.x.value}\ny: ${c.y.value}\nz: ${c.z.value}',
              style: const TextStyle(fontSize: 24),
            )),
      ),
    );
  }
}
```

## 4. Studi Kasus: Aplikasi Deteksi Guncangan

Buat aplikasi yang mendeteksi guncangan (shake) menggunakan accelerometer.

### 4.1. Logika Deteksi Guncangan

```dart
import 'package:get/get.dart';
import 'package:sensors_plus/sensors_plus.dart';
import 'dart:math';

class ShakeController extends GetxController {
  var isShaking = false.obs;

  @override
  void onInit() {
    super.onInit();
    accelerometerEvents.listen((event) {
      double magnitude = sqrt(event.x * event.x + event.y * event.y + event.z * event.z);
      if (magnitude > 15) {
        isShaking.value = true;
      } else {
        isShaking.value = false;
      }
    });
  }
}
```

### 4.2. UI Deteksi Guncangan

```dart
import 'package:flutter/material.dart';
import 'package:get/get.dart';
import 'shake_controller.dart';

class ShakePage extends StatelessWidget {
  final ShakeController c = Get.put(ShakeController());

  ShakePage({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('Deteksi Guncangan')),
      body: Center(
        child: Obx(() => Text(
              c.isShaking.value ? 'Guncangan Terdeteksi!' : 'Tidak Ada Guncangan',
              style: const TextStyle(fontSize: 28, color: Colors.red),
            )),
      ),
    );
  }
}
```

## 5. Best Practice Penggunaan Sensor

- Gunakan listener hanya saat diperlukan (misal, diaktifkan saat halaman terbuka).
- Matikan listener saat tidak digunakan untuk menghemat baterai.
- Validasi data sensor sebelum digunakan.
- Perhatikan permission di Android/iOS jika diperlukan.
